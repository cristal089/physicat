## Sobre

`PhysiCat` é um repositório criado para desenvolvimento de um jogo educativo para computador com o intuito de ensinar seus jogadores sobre alguns dos principais conceitos de Física. Seu
objetivo é o de auxiliar adolescentes que estão adentrando o Ensino Médio a fixar seus conhecimentos sobre atrito, leis de Newton e termodinâmica. Em PhysiCat o jogador é apresentado à personagem principal, uma cientista, e deve ajudá-la a vencer desafios relacionados aos conceitos citados anteriormente. O jogo é dividido em fases, cada qual com seu desafio, sendo elas compostas por plataformas e obstáculos específicos.

---

## Como baixar o jogo

Caso você queira baixar este repositório para auxiliar no desenvolvimento, isso pode ser feito pelo terminal. Será necessário utilizar o Unity.

```bash
$ git clone https://github.com/cristal089/physicat
```

Ou, se for apenas para jogar, acesse <a href=https://cristal089.itch.io/physicat>PhysiCat no itch.io</a>.
